f# FakeNFT Structure

Use this placement policy:
- `App/`: app launch, root composition, configuration, app resources.
- `Core/`: network, storage, navigation, design system, base, utilities.
- `Shared/`: reusable UI and models shared by several features.
- `Features/`: business features and their MVVM folders.
- `SupportingFiles/`: architecture notes and templates.
- `Tests/`: unit and UI tests.
